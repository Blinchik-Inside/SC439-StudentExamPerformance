# SC439-StudentExamPerformance
This is a project for CS439 (Introduction to Data Science) project at Rutgers University, focusing on Student Exam Performance Prediction.

## Dataset Used
This project uses the "Student Exam Performance Dataset Analysis" dataset uploaded to Kaggle by Muhammad Shoaib (grandmaster07): https://www.kaggle.com/datasets/grandmaster07/student-exam-performance-dataset-analysis
In no way try to claim the ownership of this dataset.
